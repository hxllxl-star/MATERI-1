HTML DASAR Part 1 HTML (HyperText Markup Language) adalah bahasa standar untuk membuat dan mendesain halaman web. HTML menggunakan markup atau tag untuk menyusun konten, seperti teks, gambar, tautan, dan lainnya. HTML terdiri dari elemen-elemen yang didefinisikan dengan tag, dan setiap tag biasanya terdiri dari tag pembuka () dan tag penutup (). Selain HTML, CSS (Cascading Style Sheets) digunakan untuk mendesain tampilan halaman web, sementara JavaScript berfungsi untuk menambahkan interaktivitas dan fungsionalitas pada halaman tersebut.
